Acerca del sitio

ICA (Icarus | Ivr Cluster Admin) fue desarrollado por Ezequiel Lovelle.

En su desarrollo se empleó exclusivamente software libre; tanto en lo que se refiere a los componentes de la aplicación en sí y los servicios que la conforman, como a las herramientas y tecnologías utilizadas en los ambientes de desarrollo. A continuación listamos, sin ningún orden particular, algunas de las piezas más importantes de software que se emplearon en el desarrollo:

  • Redis
  • Python
  • jQuery
  • Jinja
  • jQuery UI
  • Gunicorn
  • Bootstrap
  • GIT
  • Nginx
  • Node.js
  • Socket.IO
  • Modernizr
  • JQuery Pjax
  • Font Awesome
